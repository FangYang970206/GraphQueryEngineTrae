# Cypher 功能范围与缺失清单

> 本文档只记录当前 `scope.md` 范围内“已支持 / 明确不支持 / 后续可评估”的能力，不再把被收紧掉的旧语义记为已实现。

## 1. 当前支持

### 1.1 查询子句

| 功能 | 状态 |
|------|------|
| `MATCH` | ✅ |
| `RETURN` | ✅ |
| `WITH` | ✅ |
| `WHERE` | ✅ |
| `ORDER BY` | ✅ |
| `LIMIT` | ✅ |
| `UNION` / `UNION ALL` | ✅ |
| `USING SNAPSHOT` | ✅ |
| `PROJECT BY` | ✅ |

### 1.2 模式与表达式

| 功能 | 状态 |
|------|------|
| 起点带 `label` 的节点模式 | ✅ |
| 多关系类型 `|` | ✅ |
| 命名路径 | ✅ |
| 属性访问 | ✅ |
| 比较 / `AND` / `OR` / `NOT` | ✅ |
| `IS NULL` / `IS NOT NULL` | ✅ |
| `IN` / `STARTS WITH` / `ENDS WITH` / `CONTAINS` | ✅ |

### 1.3 聚合函数

| 功能 | 状态 |
|------|------|
| `count` / `count(*)` | ✅ |
| `sum` / `avg` / `min` / `max` | ✅ |

## 2. 明确不支持

以下能力不是“待补齐”，而是当前范围明确排除：

| 功能 | 状态 | 原因 |
|------|------|------|
| `OPTIONAL MATCH` | ❌ | 超出当前范围 |
| `UNWIND` | ❌ | 超出当前范围 |
| `CALL` / 存储过程 | ❌ | 超出当前范围 |
| `SKIP` | ❌ | 分页链路已收紧 |
| 变长路径 | ❌ | 解析阶段直接拒绝 |
| 单跳虚拟边 | ❌ | 违反边界约束 |
| 纯虚拟点 | ❌ | 违反边界约束 |
| 虚拟到虚拟路径 | ❌ | 违反边界约束 |
| 写语句 | ❌ | 只读设计 |

## 3. 后续可评估但未纳入范围

这些能力即使语法层面可能存在痕迹，也不应被当作“已实现”：

| 功能 | 当前结论 |
|------|----------|
| `collect` | 不在当前允许聚合函数集合 |
| `REGEXP` | 不在当前表达式支持集合 |
| `XOR` | 不在当前表达式支持集合 |
| 动态属性访问 / 列表切片 | 不在当前表达式支持集合 |
| 日期函数、统计函数、路径函数 | 未纳入当前范围 |

## 4. 收口说明

- `WHERE` 统一按变量归属下沉到相关数据源
- 第一跳虚拟边执行顺序为“外部 -> 提取 ID -> TuGraph”
- 最后一跳虚拟边执行顺序为“TuGraph -> 提取 ID -> 外部”
- 默认限制为外部计划 `1000`、纯 TuGraph 计划 `5000`
- 最终总结果上限为 `8000`
