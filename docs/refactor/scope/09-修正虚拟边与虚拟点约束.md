# 09 修正虚拟边与虚拟点约束

## 目标

让虚拟边和虚拟点的判定、校验、改写与 `scope.md` 完全一致。

## 影响文件

- `graph-query-engine/src/main/java/com/fangyang/federatedquery/rewriter/VirtualEdgeDetector.java`
- `graph-query-engine/src/main/java/com/fangyang/federatedquery/rewriter/MixedPatternRewriter.java`
- `graph-query-engine/src/main/java/com/fangyang/federatedquery/rewriter/QueryRewriter.java`

## 修改要求

- 增加“单跳虚拟边不允许”的显式校验。
- 增加“纯虚拟点不允许”的显式校验。
- 增加“虚拟到虚拟不支持”的显式校验。
- 修正 `[physical] -> [virtual] -> [physical]` 校验逻辑，按原始路径顺序判断，不能先按类型重组。
- 保留并校验 `firstHopOnly`、`lastHopOnly` 约束。

## 验收标准

- 所有违反虚拟边边界的模式都在 rewrite 前后稳定报 `VirtualEdgeConstraintException` 或统一错误。
- 合法与非法模式的判定结果与 `scope.md` 示例一致。
